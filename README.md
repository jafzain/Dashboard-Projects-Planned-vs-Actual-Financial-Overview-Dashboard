# Dashboard-Projects-Planned-vs-Actual-Financial-Overview
[PDF](ProjectsPlannedvsActual&Financial Overview.pdf)
This dashboard provides a clear view of project performance by comparing planned vs. actual revenue, expenses, timelines, and profitability. It highlights key metrics such as budget variance, cost overruns, percentage of completion (POC), and delivery status across departments and regions.
The insights help management quickly identify deviations, track progress, and make informed decisions to improve project efficiency and financial accuracy.
Key Focus Areas:
Actual vs Estimated Revenue & Expenses
Budget Variance & Cost Overrun
Profitability & POC Analysis
Project Delivery Performance
Regional & Departmental Insights
